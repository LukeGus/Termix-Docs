# Features
General how-to for some of the less obvious Termix features

## Admin Settings
On an admin user, click on your username on the bottom left, then click `Admin Settings`.

## OIDC
OIDC must be configured within admin settings after creating a local account. It cannot be created using environment variables yet. See [OIDC](https://docs.termix.site/oidc) for more.

## Host Management
Add SSH hosts and credentials within the `Host Manager` button in the top left after logging in. Here, you can create/customize hosts.

## Credential Deploy
Create a credential in the `Host Manager` in the `Add Credential` tab with an SSH key as the selected auth type. Then, go back to the `Credential Viewer` tab and click on the green arrow icon on the desired credential and follow the on-screen steps.

## Split Screen
Click on the `<|>` icon on a tab in the top bar (next to the close button). This will open the `SSH Tool Sidebar` in the split-screen tab. Follow the on-screen steps.

## SSH Tools / Command History / Snippets / Split Screen
Click on the `Hammer` icon in the top right, it will open a sidebar with tabs for all the above features.

## Server Stats
Configure a host with Server Stats enabled in `Host Manager` then visit the `Server Details` option on a host when connecting to it from the `Left Sidebar`. See [server stats](https://docs.termix.site/server-stats) for more.

## Export/Import
Visit the `Admin Settings` and go to the `Database` tab.

## SSH Tunnels
Configure a host with a tunnel configured in `Host Manager` then visit the `Server Details` option on a host when connecting to it from the `Left Sidebar`. See [tunnels](https://docs.termix.site/tunnels) for more.

## File Manager
Configure a host with File Manager enabled in `Host Manager` then visit the `File Manager` option on a host when connecting to it from the `Left Sidebar`. The file manager only works on common Linux based hosts.

## Command Palette
Double tap `Left Shift` to open.

## Support

If you need help or want to request a feature with Termix, visit the [Issues](https://github.com/Termix-SSH/Support/issues) page, log in, and press `New Issue`.
Please be as detailed as possible in your issue, preferably written in English. You can also join the [Discord](https://discord.gg/jVQGdvHDrf) server and visit the support
channel, however, response times may be longer.
